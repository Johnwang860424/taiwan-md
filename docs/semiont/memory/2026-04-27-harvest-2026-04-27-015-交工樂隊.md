## 主要研究發現

### 切入人物

林生祥（1972–）：美濃客家囝仔、交工樂隊主唱/吉他手，1998 年關掉淡水銀行帳戶只剩 57 元返鄉，因反水庫運動而開始音樂創作。

### 反直覺核心句

「這支樂隊從沒想過進入唱片業——歌是為了立法院門口的抗議人群唱的——但它最終成為台灣音樂史上最被引用的經典之一。」

### 關鍵事件時間軸

| 年份 | 事件 |
|------|------|
| 1992年底 | 美濃愛鄉協進會成立，反水庫運動開始 |
| 1998年 | 林生祥返鄉美濃（帳戶剩57元）；與鍾永豐開始創作 |
| 1999年4月 | 《我等就來唱山歌》發行 |
| 2000年 | 第11屆金曲獎奪最佳作曲人、最佳製作人；陳水扁宣布不建美濃水庫 |
| 2001年9月 | 《菊花夜行軍》發行（WTO/農村衰退主題） |
| 2002年 | 第13屆金曲獎最佳樂團；林生祥感謝土地伯公 |
| 2003年9月1日 | 解散 |
| 2016年 | 《菊花夜行軍》15週年廢棄菸樓演唱會（3,000人） |
